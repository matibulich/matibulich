## 📁 Proyectos Destacados

### [My Movie App](https://mymovieapp-2.onrender.com/)
Aplicación web para buscar y explorar información sobre películas utilizando la API de TMDB. Incluye autenticación y favoritos guardados en MongoDB.
- **Tecnologías:** React, JavaScript, Mongoose, REST API.
- [Repositorio en GitHub](https://github.com/matibulich/MyMovieApp)

### [Autosync-Dashboard (IA)](https://github.com/matibulich/autosync_dash_IA)
Controlador de stock inteligente que integra un modelo de IA para realizar consultas de inventario en lenguaje natural.
- **Tecnologías:** Node.js, Express, React, Prisma, MySQL.

### [Chat Inteligente](https://chatinteligente.netlify.app)
Aplicación de chat impulsada por el modelo de IA Llama, optimizada para funcionar incluso offline.
- **Tecnologías:** JavaScript, Web LLM.
- [Repositorio en GitHub](https://github.com/matibulich/chatOnline)

### [Autenticador JWT](https://autenticaci-n-jwt.onrender.com/)
Sistema de autenticación segura con registro, login y protección de rutas.
- **Tecnologías:** Node.js, Express, JWT.
- [Repositorio en GitHub](https://github.com/matibulich/autenticaci-n_jwt)
